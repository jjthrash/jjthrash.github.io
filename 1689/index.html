1689 - A Simple Study Application

Contains the full text of the 1689 London Baptist Confession of Faith. By Jimmy Thrasher and Ande Truman.

Features

  • Simple, straightforward navigation
  • Flip between chapters while reading them
  • Scripture references. If you're offline, you can use the built-in KJV. If you're online, it can take you to Bible Gateway
